# Myna

Myna is a React + Vite real-time messaging application focused on private
conversations, instant communication, contact management, and secure message
delivery.

The frontend production URL is currently:

```text
https://convo-5wqy.onrender.com
```

The production URL can remain during the rebrand, but public branding,
metadata, manifest content, sitemap entries, and user-facing application text
should use the Myna product name.
