# Myna

Myna is a privacy-focused real-time messaging application for private
conversations, instant communication, contact management, and secure message
delivery.

## Frontend

This frontend is built with React, TypeScript, and Vite. It provides public SEO
pages at:

- `/`
- `/features`
- `/security`
- `/privacy`
- `/about`
- `/contact`

Private application routes such as login, registration, health checks, user
workspaces, contacts, profiles, messages, and settings are excluded from the
public sitemap and use `noindex, nofollow` metadata.

## Environment

Production currently uses:

```text
VITE_PUBLIC_SITE_URL=https://convo-5wqy.onrender.com
```

Changing `VITE_PUBLIC_SITE_URL` updates dynamic canonical URLs, Open Graph URLs,
Twitter image URLs, and structured data URLs generated by the React app.

## Deployment

The app is deployed as a Render Static Site and requires the SPA fallback:

```text
Source: /*
Destination: /index.html
Action: Rewrite
```

Static files such as `/robots.txt`, `/sitemap.xml`, `/manifest.webmanifest`, and
image assets must remain directly accessible.
